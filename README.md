# oxNote

A note taking personal wiki solution to capture my life.

## Philosophy

- Keep it simple, silly, stupid
- Avoid feature creep
- Avoid cognitive load
- Minimal design overhead
- Minimal use of colors
- Everything inside a database, flat files are a useless
  abstraction.
